Seja bem-vindo(a) ao Crocodille Project

O que somos?

O Crocodille Project é uma iniciativa para compartilhar dados totalmente atualizados sobre a pandemia do COVID-19

Qual o nosso objetivo?

Nosso objetivo é trazer informação de forma coeza e clara para que as pessoas fiquem cientes do que está acontecendo ao redor do mundo sobre a gripe que atormentou nosso planeta nos últimos 2 anos

Quais ferramentas usamos?

Nós utilizamos o editor Visual Studio Code, as linguagens: HTML5, CSS3 E JavaScript

De onde vieram nosso dados?

Nosso dados vêm de uma API pública chamada Brazil API (https://covid19-brazil-api-docs.vercel.app/)
